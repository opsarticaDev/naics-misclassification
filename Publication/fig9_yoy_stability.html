Note: Analysis includes all establishments present in at least 2 consecutive years. "Stable" means the 6-digit NAICS code was unchanged between years. "Within-sector" means the code changed but the 2-digit sector remained the same. "Cross-sector" means both the code and the 2-digit sector changed. DART direction is computed by comparing BLS DART rates for the old and new NAICS codes using each year's published benchmarks. "Downward" shifts (lower DART rate) may indicate reclassification to a less hazardous industry. The 2021→2022 pair shows elevated cross-sector shifts and an excess of downward DART changes, potentially reflecting pandemic recovery reclassifications. Consecutive-year pairs dominate (97.8% of all pairs); skip-year pairs (e.g., 2021→2023) are included for establishments absent in intervening years.
